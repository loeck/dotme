## Requirements

- nodejs v10.x (https://nodejs.org/en/)
- yarn latest (https://yarnpkg.com/fr/docs/install)

## Installation

```
yarn
```

```
cp .env.example .env
```

## Development

```
yarn dev
```

## Production

```
yarn build
```

```
NODE_ENV=production yarn start
```
