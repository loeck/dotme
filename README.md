## Requirements

- nodejs v10.x (https://nodejs.org/en/)
- yarn latest (https://yarnpkg.com/fr/docs/install)

## Installation

```
yarn
```

## Development

```
cp .env.example .env
```

```
yarn build:development -w
yarn dev
```

## Production

```
yarn deploy
```
